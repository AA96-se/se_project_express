## Quick Start

### Prereqs

- Node.js LTS
- Local MongoDB running (default DB: `wtwr_db`)
- A **JWT secret** for signing tokens

### Setup

```bash
# 1) Install dependencies
npm install

# 2) Start the server (default port: 3001)
npm start
# (Optional dev script)
# npm run dev
```
